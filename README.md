## Setup and Instructions
To open this project in vscode using git in Terminal/Git Bash:

```
$ mkdir team-profile-generator
$ cd team-profile-generator
$ git clone https://github.com/DeRomuald/team-profile-generator.git
$ code .
$ npm i
$ node index.js
```
